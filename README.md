# Currency-Converter
A simple and responsive currency converter web app built using HTML, CSS, and JavaScript. It allows users to convert amounts between different currencies in real time using up-to-date exchange rates.
Features 🚀

🌐 Real-time currency conversion

🔄 Supports multiple currencies

📱 Responsive design for mobile and desktop

🎨 Clean and user-friendly UI

⚡ Built with HTML, CSS, and Vanilla JavaScript

How It Works 🛠️

Select the source currency and target currency.

Enter the amount you want to convert.

Instantly view the converted value based on live exchange rates.

Tech Stack 🧩

HTML5 – Structure

CSS3 – Styling and responsiveness

JavaScript (ES6) – Logic and dynamic updates
